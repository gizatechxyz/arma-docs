Follow these steps to start using ARMA:

## Prerequisites

- A Web3 wallet (MetaMask recommended)
- USDC or USDT on Mode Network
- Some ETH on Mode Network for gas fees

## Setup Process

1. **Connect Wallet**
   - Visit [ARMA](https://arma.gizatech.xyz)
   - Click "Connect Wallet"
   - Select your Web3 wallet

2. **Create Smart Account**
   - Click "Create Smart Account"
   - Approve the transaction
   - Wait for account deployment

3. **Deposit Funds**
   - Select the asset (USDC or USDT)
   - Enter the amount
   - Approve the token spend
   - Confirm the deposit

4. **Activate Agent**
   - Review and accept terms
   - Set your risk preferences
   - Click "Activate Agent"

Your funds will now be automatically managed by ARMA for optimal yields.
